# 第1章 Rustと計算物理学

本章は、プログラミング言語Rustを用いた計算物理学の学習を開始するための導入部です。
本章では、計算物理学の分野におけるRustの有用性を概説し、開発環境の構築手順を示します。さらに、本書を効果的に学習するための指針を提示します。

## 本章の構成

本章は、以下の内容で構成されます。

- **[なぜRustなのか？](./why-rust.md)**
  計算物理学の分野で従来使用されてきた言語と比較し、Rustが持つパフォーマンス、安全性、並列処理における利点を解説します。

- **[開発環境のセットアップ](./setup.md)**
  Rustコンパイラおよび関連ツールを導入し、簡単な物理シミュレーションのプログラムを実行するまでの一連のプロセスを通して、開発環境を構築する手順を解説します。

- **[数値計算結果の可視化](./plotting.md)**
  シミュレーション結果を視覚的に分析する重要性について述べ、その具体的な手法として、CSVファイルへの出力やRust製ライブラリの活用法を紹介します。

- **[本書の使い方](./how-to-use.md)**
  本書の構成を概観し、サンプルコードを実践しながら学習を進めるための、より詳細な指針を提示します。

本章を終えることで、読者はRustによる数値計算を行うための基本的な開発環境を構築し、本書を用いた学習を開始するための準備が整います。
