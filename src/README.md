# はじめに

本書は、Rustプログラミング言語を用いた計算物理学の実践的なチュートリアルです。数値計算の基礎から物理シミュレーションの実装までを体系的に解説し、Rustによる科学計算への入門を目的とします。

## 本書の目的

計算物理学は、現代の物理学研究において不可欠な手法です。解析的に解くことが困難な問題に対し、数値計算を用いることで具体的な答えを得られます。一方、Rustは安全性とパフォーマンスを両立したプログラミング言語として注目されています。

本書は、これら2つの分野を結びつけ、Rustを用いて計算物理学の基本的な手法を実装し、その理解を深めることを目的としています。本書を通じて、読者の皆さんがご自身で物理シミュレーションを設計・実装し、結果を分析できるようになることを目指します。

## 想定読者

本書は、以下のような読者を想定しています。

- **Rustの基本文法を理解している方**：変数、関数、構造体、トレイト、所有権など、Rustの基本的な概念を学んだことがある方を対象とします。高度な知識は不要ですが、基本的なプログラムが書ける程度の理解を前提としています。

- **学部レベルの物理学を学んだ方**：力学、電磁気学、熱力学・統計力学、量子力学といった基礎的な物理学の知識があることを前提とします。ただし、各トピックでは必要に応じて簡単な復習も行います。

プログラミングや物理学の専門家である必要はありません。むしろ、これらの分野を学びながら実践的なスキルを身につけたい学生や研究者の皆さんに最適な内容となっています。

> [!NOTE]
> **Rustをまだ学んでいない方へ**
>
> もしRustに初めて触れる場合は、本書を読む前に公式ドキュメントを一読することをお勧めします。公式ドキュメントは非常に丁寧に書かれており、Rustの基本概念を体系的に学べます。
>
> - [The Rust Programming Language（英語版）](https://doc.rust-lang.org/book/)

## 本書の特徴

本書には、以下のような特徴があります。

- **実践重視のアプローチ**：理論的な説明も大切にしつつ、実際にコードを書いて動かし、結果を確認することを重視しています。各トピックで具体的な実装例を示し、読者がご自身の手で試せるように構成しています。

- **体系的・網羅的な内容**：数値計算の基礎から始まり、常微分方程式、偏微分方程式、モンテカルロ法、並列計算、可視化まで、計算物理学の主要なトピックを幅広く扱っています。

## 本書で扱う内容

本書は、大きく分けて以下の4部構成です。

- **第1部 基礎編**：Rustと計算物理学の関係、開発環境のセットアップ、そして数値計算の基礎（浮動小数点数、配列操作、主要クレートの使い方など）を学びます。

- **第2部 数値計算手法**：科学計算の核となる技術を扱います。数値微分・積分、非線形方程式、線形代数、フーリエ解析、常微分方程式、偏微分方程式、モンテカルロ法などの数値解法を実装し、その特性を理解します。

- **第3部 物理シミュレーション**：実際の物理現象をシミュレートします。古典力学、流体力学、統計力学（イジング模型など）、量子力学といった、多様な物理分野の計算手法を学びます。

- **第4部 高度なトピック**：より実用的なシミュレーション技術を扱います。並列計算による高速化、SIMD最適化、パフォーマンス測定などを通じて、実践的なスキルを身につけます。

## 本書の使い方

本書は、基本的に最初の章から順番に読み進めることを推奨します。ただし、特定のトピックに興味がある場合は、前述の依存関係を参考に必要な章から始めることも可能です。第1部（基礎編）は以降の章で前提知識となることが多いため、一度は目を通すようにしてください。

各章は、概ね以下の流れで進みます。

1.  **概念の理解**：トピックの物理的・数学的背景を簡潔に説明します。
2.  **実装**：Rustによる具体的な実装方法を示します。
3.  **実行と可視化**：コードを実行し、得られた結果を確認します。
4.  **考察**：結果の解釈や、手法の特性について議論します。

コードは、実際に手元で動かしてみることを強く推奨します。エラーへの対処や、期待した結果が得られない場合の試行錯誤も、学習の重要なプロセスです。こうした経験を通じて、より深い理解が得られるでしょう。
