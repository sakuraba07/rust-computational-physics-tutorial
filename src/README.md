# はじめに

本書は、Rustプログラミング言語を用いた計算物理学の実践的なチュートリアルです。数値計算の基礎から、実際の物理シミュレーションまでを体系的に学ぶことで、Rustによる科学計算の入門を目指します。

## 本書の目的

計算物理学は、現代の物理学研究において不可欠な手法となっています。解析的に解くことが困難な問題に対して、数値計算を用いることで具体的な結果を得ることができます。一方、Rustは安全性とパフォーマンスを両立した新しいプログラミング言語として注目されています。

本書は、これら2つの分野を結びつけ、Rustを使って計算物理学の基本的な手法を実装し、理解を深めることを目的としています。本書を読み終えた時には、読者の皆さんが自分自身で物理シミュレーションを設計・実装し、結果を分析できるようになることを目指します。

## 想定読者

本書は以下のような読者を想定しています。

- **Rustの基本文法を理解している方**：変数、関数、構造体、トレイト、所有権などの基本的な概念に触れたことがある方を対象としています。高度なRustの知識は必要ありませんが、基本的なプログラムが書ける程度の理解があることを前提とします。

- **学部レベルの物理学を学んだ方**：力学、電磁気学、熱力学・統計力学、量子力学などの基礎的な物理学の知識があることを前提とします。ただし、各トピックについては必要に応じて簡単な復習も含めています。

プログラミングや物理学の専門家である必要はありません。むしろ、これらの分野を学びながら実践的なスキルを身につけたいと考えている学生や研究者の方々に最適な内容となっています。

## 本書の特徴

本書には以下のような特徴があります。

- **実践重視のアプローチ**：理論的な説明も大切にしながら、実際に動くコードを書き、結果を確認することを重視しています。各トピックでは具体的な実装例を示し、読者が自分の手で試せるようになっています。

- **体系的・網羅的な内容**：数値計算の基礎から始まり、常微分方程式、偏微分方程式、モンテカルロ法、並列計算、可視化まで、計算物理学の主要なトピックを幅広くカバーしています。

## 本書で扱う内容

本書は以下のように大きく4つのパートに分かれています。

- **基礎編**：Rustと計算物理学の関係、開発環境のセットアップ、そして数値計算の基礎（浮動小数点演算、配列操作、主要なクレートの使い方など）を学びます。

- **数値計算手法**：科学計算の核となる技術を扱います。数値微分・積分、線形代数、常微分方程式、偏微分方程式などの数値解法を実装し、その特性を理解します。

- **物理シミュレーション**：実際の物理現象をシミュレートします。古典力学、モンテカルロ法、統計力学（イジング模型など）、量子力学、流体力学など、多様な物理分野の計算手法を学びます。

- **高度なトピック**：実用的なシミュレーションに必要な技術を扱います。並列計算による高速化、可視化技術、高速フーリエ変換（FFT）などを通じて、より実践的なスキルを身につけます。

## 本書の使い方

本書は基本的に順番に読み進めることを推奨しますが、特定のトピックに興味がある場合は、そこから始めることも可能です。ただし、基礎編の内容は後の章で前提とされることが多いため、一度は目を通すことをお勧めします。

各章では以下のような流れで学習を進めます。

1. **概念の理解**：そのトピックの物理的・数学的背景を簡潔に説明します
2. **実装**：Rustでの具体的な実装方法を示します
3. **実行と可視化**：コードを実行し、結果を確認します
4. **考察**：結果の解釈や、手法の特性について議論します

コードは実際に手元で動かしてみることを強く推奨します。エラーに遭遇したり、期待した結果が得られなかったりすることも学習の重要な一部です。試行錯誤を通じて、より深い理解が得られるでしょう。
