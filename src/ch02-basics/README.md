# 第2章 数値計算の基礎

> [!IMPORTANT]
> **本章の前提知識**
>
> 本章の内容を理解するためには、以下の章を事前に学習しておく必要があります。
>
> - [第1章 Rustと計算物理学](../ch01-introduction/)

第1章では、Rustを用いて計算物理学を学習するための準備を行いました。本章より、その中核を成す数値計算の世界へと入っていきます。

本章で扱う内容は、以降のすべての章（第2部「数値計算手法」、第3部「物理シミュレーション」）の根幹を成す、極めて重要な基礎です。本章では、コンピュータにおける数値表現といった基本的なテーマから、効率的なベクトル・行列演算を実現するライブラリに至るまで、Rustで数値計算を行う上で必須となる知識を解説します。

## 本章の構成

- **[浮動小数点演算と誤差](./floating-point.md)**
  コンピュータにおける実数の表現形式である「浮動小数点数」の仕組みと、それに伴い必然的に生じる「丸め誤差」や「桁落ち」といった計算誤差について解説します。これらは、数値計算の結果を正しく評価する上で不可欠な知識です。

- **[配列とベクトル演算](./arrays-vectors.md)**
  複数の数値をまとめて扱うための基本的なデータ構造である、Rust標準の配列、スライス、そして`Vec<T>`について解説します。

- **[外部クレートの活用（ndarray入門）](./ndarray.md)**
  PythonにおけるNumPyのように、科学技術計算では多次元配列（ベクトルや行列）を効率的に扱うための専用ライブラリが不可欠です。本節では、Rustのエコシステムで広く利用されている`ndarray`クレートの基本的な使用法を習得します。

- **[高精度演算](./high-precision.md)**
  標準の64ビット浮動小数点数 (`f64`) では精度が不足するような特定の問題に対処するため、`double-double`演算などを用いてより高い精度を実現する手法を解説します。

本章を終えることで、読者はRustを用いて数値を正確かつ効率的に扱うための基礎知識とツールを習得できます。この土台の上に、第3章以降で様々な数値計算アルゴリズムを構築していきます。
