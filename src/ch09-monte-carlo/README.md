**モンテカルロ法 (Monte Carlo method)** は、乱数を用いた試行を繰り返すことで近似解を得る数値計算手法の総称です。

## モンテカルロ法の重要性

第3章で学んだような決定論的な積分手法（台形則など）は、1次元や2次元では非常に高精度ですが、次元が増える（変数の数が増える）と計算量が指数関数的に増加してしまいます（**次元の呪い**）。

一方、モンテカルロ法の誤差は次元に依存せず、試行回数 $N$ に対して $1 / sqrt(N)$ で減少します。そのため、以下のような場面で威力を発揮します。

- **多次元積分**: 数十〜数百次元の積分計算。
- **統計力学**: 多数の粒子からなる系の平均的な状態（エネルギーや磁化など）の推定。
- **量子力学**: 経路積分や多体問題。

## 本章の構成

1. **[乱数生成](./random-numbers.md)**
   計算の基礎となる高品質な疑似乱数の生成方法と、特定の分布（正規分布など）に従う乱数の作り方を学びます。

2. **[モンテカルロ積分](./integration.md)**
   乱数を使って積分値を求める最も基本的な手法を学びます。

3. **[重点サンプリング](./importance-sampling.md)**
   効率的に計算を行うために、サンプリングに「重み」をつけることで誤差を減らす手法を学びます。

4. **[マルコフ連鎖モンテカルロ法 (MCMC)](./mcmc.md)**
   現代の物理シミュレーションの主役である、マルコフ連鎖を用いたサンプリング（メトロポリス法など）の基礎を学びます。
