# 常微分方程式

> [!IMPORTANT]
> **この章を読む前に**
>
> この章を読むには、以下の章を先に読んでおく必要があります。
>
> - **必須**：[第2章: 数値計算の基礎](../ch02-basics/)
> - **必須**：[第3章: 数値微分と数値積分](../ch03-calculus/)

物理法則の多くは、時間や空間に対する変化率（微分）を含む**微分方程式 (Differential Equations)** として記述されます。

$$ m dv(x, t, 2) = F(x, v, t) quad (text("ニュートンの運動方程式")) $$

$$ L dv(I, t) + R I = V(t) quad (text("回路方程式")) $$

本章では、時間変数 $t$ の関数である $x(t)$ に対する**常微分方程式 (Ordinary Differential Equation, ODE)** の初期値問題を数値的に解く手法を学びます。

## 本章の構成

1. **[オイラー法](./euler.md)**
   最も単純な解法であるオイラー法を通じて、微分方程式を数値的に解く（時間発展させる）基本的な考え方を学びます。

2. **[ルンゲ＝クッタ法](./runge-kutta.md)**
   実用的な標準手法である4次のルンゲ＝クッタ法 (RK4) を学びます。精度と計算コストのバランスが良く、最も広く使われている手法の一つです。

3. **[適応型刻み幅制御](./adaptive-step.md)**
   誤差の大きさに応じて時間刻み $Delta t$ を自動的に調整し、効率よく高精度な解を得る手法（ルンゲ＝クッタ＝フェールベルグ法など）を紹介します。

4. **[境界値問題](./boundary-value.md)**
   初期値問題とは異なり、区間の両端で条件が与えられる問題（例：両端が固定された弦の振動など）を解く「シューティング法」について触れます。
